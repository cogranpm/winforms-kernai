allow user to enter and execute f# script dynamically
so user can practice f# coding concepts
create exercises for user to type in:
exercise: some text explaining what is required
hint: in case user is stuck and needs a hint
solution: full text of the solution

    3. a console window to show the results from the fsi process
    4. a hint text
    
need 3 button interactions
    1. a button to execute the script
    2. a hint button if stuck shows help
    3. a button for the TOC item we are working on
    4. a button for the exercise being undertaken
